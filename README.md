## Installation
> 본 프로젝트에서 제공하는 모든 기능을 사용하기 위해서 다음 명령어를 통해 설치합니다.
  
**Requests**

    pip install requests

**Beautifulsoup**

    pip install bs4

**Tkinter**

    pip install tkinter

**Matplotlib**

    pip install matplotlib

**Openpyxl**

    pip install openpyxl

**Konlpy**

    pip install konlpy

**JPype1**

    pip install "JPype1<1.1"
